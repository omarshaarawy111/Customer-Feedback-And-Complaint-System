# Customer Feedback And Complaint System

> ⚠️ This is a **simplified demo version** of an internal SEO automation system.  
> All proprietary code, credentials, and platform-specific logic have been removed to comply with confidentiality policies.

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://python.org)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)](https://streamlit.io)
[![Plotly](https://img.shields.io/badge/Plotly-5.15+-green.svg)](https://plotly.com)

A comprehensive **Streamlit-based application** for analyzing data, generating interactive visualizations, and creating professional HTML reports. Built for modern social media analytics and brand monitoring workflows.

## ✨ Key Features

- 📊 **Multi-Format Data Support**: Upload CSV/Excel files for analysis
- 🔍 **Advanced Filtering**: Filter by year, quarter, month, brand, country, type, and contact method
- 📈 **Interactive Visualizations**: Dynamic Plotly charts (bar, pie, line, scatter)
- 📋 **Professional Reports**: Generate and export comprehensive HTML reports
- 🎨 **Customizable Themes**: Multiple color schemes and branding options
- 🔐 **Secure Access**: Built-in authentication system
- 🌐 **Multi-Brand Support**: Designed for enterprise complaint system workflows

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/omarshaarawy111/Customer-Feedback-And-Complaint-System.git
cd Customer-Feedback-And-Complaint-System

# Create virtual environment
python -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate    # Windows

# Install dependencies
pip install -r requirements.txt

# Launch application
streamlit run src/main.py
```

## 🔍 Data Processing Capabilities

**Supported Data Sources:**
- CSV files (UTF-8 encoding)
- Excel files (.xlsx, .xls)
- Multiple sheets per workbook
- Large datasets (100K+ records)

**Analysis Dimensions:**
- **Temporal**: Year, quarter, month, day
- **Geographic**: Country, region, city-level
- **Brand**: Multi-brand comparison
- **Channel**: Platform-specific insights
- **Sentiment**: Positive/negative/neutral classification
- **Engagement**: Likes, shares, comments, reach
